Help Documentation

Argument Creation

Generate Argument: Create a new argument using AI. Arguments can be contentions, rebuttals, or general collections of cards. The AI will generate a sequence of uncut cards with taglines and sources.

Create Blank Argument: Start with a blank argument containing one empty card. Useful for manually building arguments without AI assistance.

Argument Management

Rename: Change the name/query of an existing argument. The argument name is displayed in the argument list.

Delete: Permanently remove an argument and all its cards from storage.

Revise Argument: Use AI to modify the current argument based on a revision prompt. The AI can add, remove, or modify cards while preserving cards. Type your revision instructions. Note that the AI only has access to the current contention and your current revision instructions, not previous instructions.

Card Operations

Add Card: Insert a new blank card below the current card (or at the beginning if the argument has no cards).

Cut Card: Use AI to analyze the source article and extract relevant paragraphs (with contiguous paragraphs and paragraph integrity) with proper formatting. The AI will create a citation based on your 'Citation Format' setting and body text with emphasis, underlining, and highlighting applied. Requires a tagline and source URL before cutting.

Cut All Cards: Automatically cut all uncut cards in the argument

Copy: Copy a single card's content with style tags ({tag}, {cite}, {highlight}, {emphasize}, {underline}) to your clipboard. After pasting into Microsoft Word, use this macro to format the style tags according to your debate format preferences.

Copy All: Copy all cards in the argument with style tags.

Uncut: Remove the citation and body text from a cut card, returning it to uncut state. The tagline and source are preserved.

Delete: Permanently remove a card from the argument.

Move: Click and drag the Move button to reorder cards within the argument.

Card Manual Editing

Tagline: Click the tagline area to edit it inline. Press Enter to save or Escape to cancel. The tagline is required before cutting a card.

Source (Uncut Cards): Click the source area to edit the URL. The source is where the model looks for content when cutting cards

Citation (Cut Cards): Click the citation to edit it manually. Editing the citation never changes the source URL

Body Text: Click the body text area to manually cut a card. Apply underlining (Un), emphasis (Em), or highlighting (Hi) to selected text. Press Enter to save or Escape to cancel.

Settings

Model: Select which AI model to use for generation. We currently only support Google's Gemini models with no plans to expand.

Highlight/Emphasis/Underline/Font Sizes: Choose formatting options for the various styles in a card

Citation Format: Customize how citations are generated when the AI cuts a card

Reset to Defaults: Restore all settings to their original values.

Data Storage

All arguments and settings are stored locally in your browser using localStorage. To backup your work, you can use your browser's developer tools to export localStorage data. A way to sync arguments on different devices is in early stages of development.

Platform Support

Aristeia is designed for laptops/desktops, but most features should work on tablets and smartphones as well.
